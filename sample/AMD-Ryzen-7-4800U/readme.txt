1. How to build and execute OpenEL library and sample application on PC.

$ cd sample/simMAC

1.1 How to build OpenEL library.

$ make openel

A library file 'libopenel.a' will be built.

1.2 How to build a sample application.

$ make sample

1.3 how to execute a sample application.

$ ./sample
or
$ make
$ ./sample
